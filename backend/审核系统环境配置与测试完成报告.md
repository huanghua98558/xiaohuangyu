## 🔧 故障排查

### 问题 1: 代理服务未启用

**现象**:
```
[WARN] 代理服务未启用或 API 地址未配置
```

**原因**: 环境变量未加载

**解决**:
```bash
# 1. 检查 .env 文件
cat .env | grep PROXY_

# 2. 重启 PM2 服务
pm2 restart backend

# 3. 验证
pm2 logs backend | grep PROXY
```

### 问题 2: 上传失败

**现象**:
```
上传失败：无法访问目录
```

**原因**: 目录不存在或权限不足

**解决**:
```bash
sudo mkdir -p /var/www/xiaohuangyu/uploads/test
sudo mkdir -p /var/www/xiaohuangyu/uploads/audit
sudo chmod -R 755 /var/www/xiaohuangyu/uploads
```

### 问题 3: 触发验证码

**现象**:
```
[WARN] 触发验证码
```

**原因**: 代理 IP 不可用

**解决**:
```bash
# 等待服务重启后自动获取 IP
# 或手动补充
node -e "
import proxyPoolService from './src/services/ai/proxyPoolService.js'
await proxyPoolService.refillIPs(10)
"
```
