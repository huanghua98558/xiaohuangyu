### 代理 IP 池工作原理

1. **自动补充**: 当池中 IP 数量 < 500 时，自动从 API 获取
2. **智能轮换**: 每次请求使用不同的 IP
3. **失败检测**: 连续失败 3 次或失败率 > 30% 时标记为不可用
4. **自动告警**: 当可用 IP < 50 时发送告警

## 4. 故障排查

### 问题 1: 上传失败 "只允许上传图片文件"

**原因**: 文件格式不正确

**解决**:
```bash
# 检查文件类型
file screenshot1.jpg

# 如果是其他格式，转换为 JPG
sips -s format jpeg screenshot1.png --out screenshot1.jpg
```

### 问题 2: 上传失败 "无法找到目录"

**原因**: 上传目录不存在或权限不足

**解决**:
```bash
# 创建目录
sudo mkdir -p /var/www/xiaohuangyu/uploads/test
sudo mkdir -p /var/www/xiaohuangyu/uploads/audit

# 设置权限
sudo chmod -R 755 /var/www/xiaohuangyu/uploads
```

### 问题 3: 链接验证触发验证码

**原因**: 代理 IP 不可用或已失效

**解决**:
```bash
# 检查代理 IP 池状态
cd /Users/huanghua/Desktop/App/xiaohuangyu1/backend
node -e "
import proxyPoolService from './src/services/ai/proxyPoolService.js'
const stats = await proxyPoolService.getStats()
console.log('IP 池统计:', stats)
"

# 如果 IP 数量不足，手动补充
node -e "
import proxyPoolService from './src/services/ai/proxyPoolService.js'
await proxyPoolService.refillIPs(10)
console.log('IP 补充完成')
"
```

### 问题 4: OCR 识别置信度低

**原因**: 截图质量不佳

**建议**:
- 使用高清截图
- 确保评论区清晰可见
- 避免截图过暗或过亮
- 确保文字完整无遮挡

### 问题 5: 代理 IP 池为空

**检查配置**:
```bash
# 检查环境变量
cat .env | grep PROXY_

# 确认配置正确
PROXY_ENABLED=true  # 必须为 true
PROXY_API_URL=...   # 必须有效
```

**手动获取 IP**:
```bash
curl "https://share.proxy.qg.net/get?key=NHR9ADJV&num=10&format=txt"
```
