# 📸 如何保存截图到测试目录

## 方法 1：从对话中保存（推荐）

### 在 Trae 对话中：

1. **找到您刚才发送的两张截图**
   - 第 1 张：视频播放页（丁丁呀在武功山的视频）
   - 第 2 张：评论区截图

2. **右键点击图片**

3. **选择"下载"或"保存图片"**

4. **保存到以下位置**：
   ```
   第 1 张保存为：/tmp/xiaohuangyu-test/screenshot1.jpg
   第 2 张保存为：/tmp/xiaohuangyu-test/screenshot2.jpg
   ```

---

## 方法 2：使用 Finder（Mac）

1. 打开 Finder
2. 按 `Cmd + Shift + G` 打开"前往文件夹"
3. 输入：`/tmp/xiaohuangyu-test`
4. 将下载的图片拖入此文件夹
5. 重命名为 `screenshot1.jpg` 和 `screenshot2.jpg`

---

## 方法 3：使用命令行

如果您知道图片的 URL 或已经下载到下载文件夹：

```bash
# 从下载文件夹复制
cp ~/Downloads/screenshot1.jpg /tmp/xiaohuangyu-test/screenshot1.jpg
cp ~/Downloads/screenshot2.jpg /tmp/xiaohuangyu-test/screenshot2.jpg

# 或者如果您知道图片 URL
curl -o /tmp/xiaohuangyu-test/screenshot1.jpg "图片 URL"
curl -o /tmp/xiaohuangyu-test/screenshot2.jpg "图片 URL"
```

---

## 验证文件是否保存成功

运行以下命令：

```bash
ls -la /tmp/xiaohuangyu-test/
```

**预期输出**：
```
total 8
drwxr-xr-x   2 huanghua  wheel    64  3 21 16:05 .
drwxrwxrwt  81 root      wheel  2592  3 21 16:09 ..
-rw-r--r--   1 huanghua  wheel  123456  3 21 16:10 screenshot1.jpg
-rw-r--r--   1 huanghua  wheel  234567  3 21 16:10 screenshot2.jpg
```

---

## 保存完成后

运行自动化测试：

```bash
cd /Users/huanghua/Desktop/App/xiaohuangyu1/backend
node run-full-test.js
```
